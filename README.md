# UIHealthbars_18
Создана универсальная система для отображения здоровья, чтобы в будущем применять в других проектах.

На сцене расположены 2 кнопки, которые симулируют получение урона и лечение. Обрабатываю текущее здоровье через ранее реализованный класс статы (Stats). 

Реализованы три индикатора, которые отображают текущее значение одного экземпляра здоровья:

Текстовый. На экране отображается текст в формате Текущее здоровье/Максимальное здоровье.

Два других индикатора реализованы помощью компонента Slider:
Бар здоровья. Степень заполненности бара отражает отношение текущего здоровья к максимальному.
Плавный бар здоровья. Тоже самое, что и предыдущий бар, однако отображает все изменения с эффектом плавности.


После выполнения задачи, импортирова систему отображения здоровья в свой пустой проект Unity как пользовательский пакет (Unity Package), чтобы использовать её в следующих разработках.
